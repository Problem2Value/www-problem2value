Maturity levels from problem to value
We firmly believe everything starts with identifying a problem worth solving. The journey towards delivering value is NOT a straight line.
We use a blend of the best aspects of leading innovation methodologies like Design Thinking, Lean Startup, Google-Venture’s Design Sprint and Agile / Scrum.

Prioritize: brings transparency in collecting, discussing and prioritising which problems are good candidates for further exploration.

Frame: provides in-depth problem space exploration and opportunity definition.

Build: ideate and develop a solution to the problem.

Evaluate: by continuously monitoring key metrics against targets and identify opportunities for improvement.

Stages
We can help at any stage of the problem solving process
Prioritize
  • Become confident in discovering problems.
  • Establish an environment where it is easy to share problems.
  • Transparently discuss and prioritise problems, NOT solutions.
Frame
  • Learn more about the problem.
  • Translate collected research data into insights.
  • Identify early adopters/personas.
  • Define a clear opportunity area.
Build
  • Ideate possible solutions.
  • Design a concept and build a prototype to validate high risk assumptions.
  • Build an MVP iteratively using Scrum.
Evaluate
  • Onboard a number of early adopters to co-create the solution.
  • Test with real users/customers to collect evidence.
  • Monitor and tweak the solution using actionable metrics.
